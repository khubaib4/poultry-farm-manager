BACKUP FILE FORMAT:
- Filename: `poultry_backup_{YYYY-MM-DD_HH-mm-ss}.db` or `.zip`
- Contains: SQLite database file
- Optional: metadata.json with backup info

BACKUP METADATA:
```typescript
interface BackupMetadata {
  version: string;          // App version
  createdAt: string;        // ISO timestamp
  dbVersion: number;        // Schema version
  stats: {
    farms: number;
    flocks: number;
    dailyEntries: number;
    expenses: number;
    vaccinations: number;
  };
}
```

AUTO-BACKUP SETTINGS:
```typescript
interface AutoBackupSettings {
  enabled: boolean;
  frequency: 'daily' | 'weekly';
  time: string;             // HH:mm format
  location: string;         // Directory path
  retention: number;        // Number of backups to keep
  lastBackup: string | null;
  nextBackup: string | null;
}
```
