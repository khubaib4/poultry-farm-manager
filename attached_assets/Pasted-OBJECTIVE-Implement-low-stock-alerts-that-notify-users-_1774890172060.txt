ALERT SEVERITY LEVELS:
- Critical (Red): quantity = 0 OR quantity < 25% of threshold
- Warning (Yellow): quantity <= threshold but > 25%
- Info (Blue): expiring within 7 days

ALERT DATA STRUCTURE:
```typescript
interface Alert {
  id: string;
  type: 'low_stock' | 'expiring' | 'vaccination_due';
  severity: 'critical' | 'warning' | 'info';
  title: string;
  message: string;
  referenceId: number; // item ID or flock ID
  createdAt: Date;
  isDismissed: boolean;
  actionUrl?: string;
}
```

UI/UX REQUIREMENTS:
- Alerts should be non-intrusive but visible
- Critical alerts more prominent
- Easy to dismiss without losing data
- Quick actions to resolve (link to relevant page)
- Badge counts update in real-time
- Accessibility: proper ARIA labels

OUTPUT: Low stock items trigger alerts shown on dashboard and in header dropdown. Users can dismiss alerts and see alert history. Thresholds are configurable per item.
