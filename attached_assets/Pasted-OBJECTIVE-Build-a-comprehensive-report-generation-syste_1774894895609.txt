REPORT DATA STRUCTURES:
```typescript
interface DailyReport {
  date: string;
  farm: FarmInfo;
  flocks: FlockDailySummary[];
  totals: {
    birds: number;
    eggsGradeA: number;
    eggsGradeB: number;
    eggsCracked: number;
    eggsTotal: number;
    deaths: number;
    feedKg: number;
    revenue: number;
  };
  notes: string[];
}

interface WeeklyReport {
  startDate: string;
  endDate: string;
  farm: FarmInfo;
  dailyData: DailyTotals[];
  weeklyTotals: Totals;
  averages: {
    eggsPerDay: number;
    mortalityRate: number;
    feedPerBird: number;
  };
}
```

UI/UX REQUIREMENTS:
- Clean, professional report layouts

- Loading states while generating

OUTPUT: Users can generate various reports, preview them in the app, and prepare them for export. Report templates are professional and comprehensive.
