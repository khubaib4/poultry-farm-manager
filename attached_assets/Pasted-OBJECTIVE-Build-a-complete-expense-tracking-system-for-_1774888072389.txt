OBJECTIVE: Build a complete expense tracking system for farm operations

REQUIREMENTS:
- Record expenses by category (feed, medicine, labor, utilities, equipment, misc)
- Track expense date, amount, description, supplier
- View expenses list with filters
- Edit and delete expenses
- Monthly expense summary

FILES TO CREATE/MODIFY:

   - Route: /farm/expenses
   - Header with "Add Expense" button
   - Filter bar:
     * Date range picker (default: current month)
     * Category dropdown (all, feed, medicine, labor, utilities, equipment, misc)
     * Search by description
   - Expense list/table:
     * Columns: Date, Category, Description, Amount (PKR), Supplier, Actions
     * Sort by date (newest first)
     * Pagination if many records
   - Summary card at top showing:
     * Total expenses for selected period
     * Breakdown by category (mini bar chart or list)

VALIDATION RULES:
- Amount: required, positive number, max 10,000,000 PKR
- Date: required, cannot be future date
- Description: required, 3-200 characters
- Category: required, must be one of the defined categories

UI/UX REQUIREMENTS:
- Large "Add Expense" button for quick entry
- Category icons for visual recognition
- Amount formatted as PKR with commas (e.g., PKR 25,000)
- Confirm before delete
- Toast notifications for success/error
- Empty state when no expenses

OUTPUT: Farm users can record, view, edit, and delete expenses. Expenses are categorized and summarized. Filter and search functionality works.
