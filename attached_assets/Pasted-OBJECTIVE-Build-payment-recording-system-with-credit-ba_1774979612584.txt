OVERDUE CALCULATION:
```typescript
const isOverdue = balanceDue > 0 && new Date(dueDate) < new Date();
const daysOverdue = isOverdue 
  ? Math.floor((new Date() - new Date(dueDate)) / (1000 * 60 * 60 * 24))
  : 0;
```

DUE SOON = Due within 7 days

PAYMENT RECORDING LOGIC:
```typescript
// Record payment
const newPaidAmount = sale.paidAmount + paymentAmount;
const newBalance = sale.totalAmount - newPaidAmount;
const newStatus = newBalance <= 0 ? 'paid' : 'partial';

// Update sale
await updateSale(saleId, {
  paidAmount: newPaidAmount,
  balanceDue: newBalance,
  paymentStatus: newStatus
});
```

VALIDATION RULES:
- Payment amount > 0
- Payment amount <= balance due
- Payment date required
- Payment method required
