REVENUE CALCULATION LOGIC:
- Daily Revenue = (Grade A eggs × Grade A price) + (Grade B eggs × Grade B price) + (Cracked eggs × Cracked price)
- Use the egg price that was active on each specific date (from egg_prices table, based on effective_date)
- If no price set for a date, use the most recent price before that date
- If no price exists at all, show warning and use 0

UI/UX REQUIREMENTS:

- Empty state if no data for period
- Quick date range buttons: Today, This Week, This Month, Last Month

OUTPUT: Farm dashboard shows revenue calculated from egg production. Daily, weekly, monthly revenue summaries available. Revenue breakdown by egg grade is visible.
